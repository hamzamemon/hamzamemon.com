1. Install the Gatsby CLI if not already installed

   ```sh
   npm install -g gatsby-cli
   ```

2. Clone the project

   ```sh
   git clone https://github.com/hamzamemon/hamzamemon.com.git
   cd hamzamemon.com
   ```

3. Install dependencies

   ```sh
   npm install
   ```
